VidBoard

In this personal project I decided that I want to start working towards creating a social game which will be able to simulate a tabletop like experience, in the sense that the players can see each other as part of the game. To achieve this I used the Agora SDK to implement a voice and vide chat demo. I studied the SDK and created both a windows and an android build. Once downloaded the user can create a room with a custom name or just tap join, to join the default one and see and hear all the other users in that room.

Details

  • Engine: Unity(C#)
  • Team Members: 1
  • Responsibilities: Research, implementations, testing
  • Duration: 70 Hours
